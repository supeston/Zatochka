# Zаточка - Профессиональная заточка инструмента

Современный лендинг для профессиональной мастерской по заточке инструментов «Zаточка». Проект создан с фокусом на плавные анимации, стильный индустриальный дизайн и высокую производительность.

## Особенности
- **Стек:** React, Vite, Tailwind CSS v4.
- **Анимации:** GSAP (включая ScrollTrigger) для параллаксов, микроинтеракций и плавных появлений.
- **Дизайн:** Индустриальная тема (графит и бирюза) с эффектами металлического блеска и бликов.
- **Адаптивность:** Полностью отзывчивый дизайн для мобильных устройств.
- **Производительность:** Ленивая загрузка компонентов (`React.lazy`), мемоизация (`React.memo`) и 60 FPS анимации.
- **Статика:** Подготовлено для размещения на GitHub Pages (base: "./").

## Разработка локально
1. Установите зависимости:
`npm install`

2. Запустите сервер для разработки:
`npm run dev &`

## Сборка и деплой на GitHub Pages
Проект настроен для статической публикации (в `vite.config.js` указано `base: "./"`).

1. Соберите проект:
`npm run build`

2. Содержимое папки `dist` полностью готово к загрузке в репозиторий GitHub Pages. Можно также протестировать скомпилированный билд локально:
`npm run preview &`
