# Zаточка - Профессиональная заточка инструмента

Современный лендинг для профессиональной мастерской по заточке инструментов «Zаточка». Проект создан с фокусом на плавные анимации, стильный индустриальный дизайн и высокую производительность.

## Особенности
- **Стек:** HTML5, Tailwind CSS v4 (через CDN), GSAP (ScrollTrigger).
- **Никакой сборки:** Проект состоит из одного файла `index.html` и папки с ассетами (`videos/`). Он не требует Node.js, Vite или React.
- **Анимации:** GSAP для параллаксов, микроинтеракций и плавных появлений.
- **Дизайн:** Индустриальная тема (графит и бирюза) с эффектами металлического блеска и бликов.
- **Адаптивность:** Полностью отзывчивый дизайн для мобильных устройств.
- **Статика:** Подготовлено для размещения на GitHub Pages прямо из корня репозитория.

## Разработка локально и деплой
1. Склонируйте репозиторий.
2. Откройте файл `index.html` в любом браузере двойным щелчком, либо используйте расширение "Live Server" в VSCode.
3. Никаких команд для сборки или установки зависимостей вводить не нужно!
4. Для деплоя на GitHub Pages просто загрузите файлы `index.html` и папку `videos/` в репозиторий и включите GitHub Pages для ветки `main` (папка `/root`).
